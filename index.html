LactoSense - Milk Quality Prediction

Team ID: MITADTHW080 | Team Name: LactoSense

PROJECT OVERVIEW

Our project focuses on using artificial intelligence and image processing to assess milk quality. By analyzing the pH level of raw and processed milk via colorimetric tests, we predict the microbial content and freshness of milk.

The process begins with capturing images of pH test strips using a smartphone camera. These images are processed using Python libraries to extract the color, which helps determine whether the milk is fresh, spoiling, or spoiled.

Using machine learning models like SVM and Random Forest, we predict milk quality, freshness duration, and microbial count. This innovative approach aims to provide real-time milk quality insights.
